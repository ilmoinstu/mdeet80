# MDEET 기출 중요 개념 암기카드

## 📚 소개
MDEET 기출 중요 개념 80문을 학습할 수 있는 플립카드 암기 앱입니다.

## ✨ 주요 기능
- 📱 **PWA 지원**: 모바일 기기에 앱으로 설치 가능
- 🌓 **다크/라이트 테마**: 시스템 설정에 따라 자동 전환
- 💾 **로컬 저장**: 학습 데이터를 브라우저에 저장
- 📲 **모바일 최적화**: 반응형 디자인으로 모든 기기에서 사용 가능
- 🔄 **플립 애니메이션**: 부드러운 카드 뒤집기 효과
- 📊 **학습 진행 상황**: 현재 카드 위치 표시

## 🎯 포함된 내용
- **총 80개 문제**: MDEET 기출 중요 개념
- **앞면**: 질문
- **뒷면**: 답변

## 🚀 사용 방법

### 웹 브라우저에서 사용
1. 웹사이트 접속
2. "학습 시작하기" 버튼 클릭
3. 카드를 클릭하여 답변 확인
4. 이전/다음 버튼으로 카드 이동

### 모바일 앱으로 설치 (Android)
1. Chrome 브라우저에서 웹사이트 접속
2. 메뉴(⋮) → "홈 화면에 추가" 선택
3. 설치 후 앱처럼 사용 가능

### 모바일 앱으로 설치 (iOS)
1. Safari 브라우저에서 웹사이트 접속
2. 공유 버튼(□↑) → "홈 화면에 추가" 선택
3. 설치 후 앱처럼 사용 가능

## 🛠️ 기술 스택
- **순수 HTML/CSS/JavaScript**: 빌드 도구 불필요
- **PWA**: Progressive Web App 기술
- **Service Worker**: 오프라인 지원
- **Local Storage**: 데이터 저장

## 📁 파일 구조
```
mdeet_app_deploy/
├── index.html          # 메인 앱 파일 (80개 MDEET 데이터 포함)
├── manifest.json       # PWA 매니페스트
├── sw.js              # Service Worker
├── icon-192.png       # 앱 아이콘 (192x192)
├── icon-512.png       # 앱 아이콘 (512x512)
└── .nojekyll          # GitHub Pages 설정
```

## 🌐 GitHub Pages 배포 방법
1. GitHub에 새 리포지토리 생성
2. `mdeet_app_deploy` 폴더의 모든 파일 업로드
3. Settings → Pages → Source를 "main" 브랜치로 설정
4. 배포된 URL로 접속
